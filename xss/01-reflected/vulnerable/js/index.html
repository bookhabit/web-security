취약점

location.search의 파라미터를 innerHTML에 직접 삽입

PHP 서버 없이도 반사형 XSS가 발생하는 클라이언트사이드 시나리오입니다.

포인트: 서버 로그에는 아무것도 안 남습니다.

여기에 출력됩니다

공격 URL 예시

  • index.html?name=<img src=x onerror=alert(document.domain)>
  • index.html?name=<svg onload=alert(document.cookie)>
  • index.html?name=<script>alert(1)</script>
