[CSP 적용] XSS 시도해도 차단

✅ CSP 적용: script-src 'self'
→ inline script, 외부 도메인 script 모두 차단
DevTools → Console 탭에서 CSP 위반 에러 확인

적용된 CSP 정책

default-src 'self' script-src 'self'

→ 현재 도메인의 .js 파일만 허용. inline script 불가.

무엇이 차단되는가?

  • ❌ <script>alert(1)</script> → inline script 차단
  • ❌ <img onerror=alert(1)> → inline event handler 차단
  • ❌ <script src="http://evil.com/x.js"> → 외부 출처 차단
  • ✅ <script src="./safe.js"> → 동일 출처 파일만 허용
