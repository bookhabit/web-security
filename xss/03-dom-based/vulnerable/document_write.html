[취약] DOM XSS - document.write()

⚠️ 이 페이지는 의도적으로 취약하게 만들어진 학습용 앱입니다.

시나리오

URL 파라미터 ?q= 값을 document.write()로 HTML에 직접 씁니다.

취약 코드 (xss.md 예시): document.write("<p>URL: " + document.location + "</p>")

공격 페이로드 예시

  • URL: document_write.html?q=<img src=x onerror=alert(1)>
  • URL: document_write.html?q=<script>alert(document.domain)</script>

location 전체를 write할 때도 취약: URL에 fragment 조작
