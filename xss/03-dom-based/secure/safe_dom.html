❌ 취약 (innerHTML) element.innerHTML = userInput document.write(input) eval(input)

방어 원리

  1. textContent: 문자열을 HTML이 아닌 텍스트 노드로 삽입 → 태그가 파싱되지 않음
  2. createElement: HTML 문자열 없이 DOM 노드를 직접 생성
  3. eval 금지: JSON 파싱은 JSON.parse(), 연산은 명시적 로직으로 처리
  4. DOMPurify: 어쩔 수 없이 HTML을 렌더링해야 할 때 사용 (sanitize)
